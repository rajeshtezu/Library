## WebRTC Debugging in Chrome

During active WebRTC session open `chrome://webrtc-internals`. It will list API traces (logs).

## Debugging in Firefox:

**Steps**

- Install `DevTools Media panel` addons
- Run WebRTC application
- Go to `tools` -> `browser tools` -> `web developer tools` -> `media-webrtc`

Or

- Open developer console and select `media-webrtc` tab.
